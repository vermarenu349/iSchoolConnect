LIST OF INLINE ELEMENTS

An inline element does not start on a new line. An inline element only takes up as much width as necessary.

Tag Name Information About Tag
Anchor The a tag defines a hyperlink, which is used to link from one page to another.
Abbreviation The abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
Bold The b tag specifies bold text without any extra importance.
Bi-Directional Override(bdo) The bdo tag is used to override the current text direction.
Break The br tag inserts a single line break.
Button The button tag defines a clickable button.
Cite The tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
Code The code tag is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
Definition element It specifies a term that is going to be defined within the content. The nearest parent of the tag must also contain the definition/explanation for the term.
Emphasized The em tag is used to define emphasized text. The content inside is typically displayed in italic.
italic The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
Image The img tag is used to embed an image in an HTML page.
Input The input tag specifies an input field where the user can enter data.
Keyboard Input The tag is used to define keyboard input. The content inside is displayed in the browser's default monospace font.
Label The
Map The map tag is used to define an image map. An image map is an image with clickable areas.
Object The object tag defines a container for an external resource. The external resource can be a web page, a picture, a media player, or a plug-in application.
Output The output tag is used to represent the result of a calculation (like one performed by a script).
q The q tag defines a short quotation.
samp The samp tag is used to define sample output from a computer program. The content inside is displayed in the browser's default monospace font.
script The script tag is used to embed a client-side script (JavaScript). The script element either contains scripting statements, or it points to an external script file through the src attribute.
Select The select element is used to create a drop-down list. The select element is most often used in a form, to collect user input.
Small The small tag defines smaller text (like copyright and other side-comments).
Span The span tag is an inline container used to mark up a part of a text, or a part of a document.
Strong The strong tag is used to define text with strong importance. The content inside is typically displayed in bold.
sub The sub tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.
sup The sup tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like WWW[1].
Textarea The textarea tag defines a multi-line text input control. The textarea element is often used in a form, to collect user inputs like comments or reviews.
Time The time tag defines a specific time (or datetime).
var The var tag is used to defines a variable in programming or in a mathematical expression. The content inside is typically displayed in italic.
