LIST OF BLOCK ELEMENTS

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).

Tag Name Information About Tag
Address The address tag defines the contact information for the author/owner of a document or an article.
Article Article is a HTML5 semantic element, similar to section and header. It is most commonly used to contain information that may be distributed independently from the rest of the site or application it appears in
Aside The aside tag is used to describe the main object of the web page in a shorter way like a highlighter.
blockquote The blockquote tag specifies a section that is quoted from another source.
Canvas The canvas tag in HTML is used to draw graphics on a web page using JavaScript.
dd Describes a term in a description list.
div The div tag defines a division or a section in an HTML document.
dl The dl tag defines a description list.
dt The dt tag defines a term/name in a description list.
Fieldset The fieldset tag is used to group related elements in a form.
Figcaption The figcaption tag defines a caption for a figure element.
figure The figure tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
footer The footer tag defines a footer for a document or section.
form The form tag is used to create an HTML form for user input.
H1 - H6 The h1 to h6 tags are used to define HTML headings. h1 defines the most important heading. h6 defines the least important heading.
header The header element represents a container for introductory content or a set of navigational links.
hr The hr element is most often displayed as a horizontal rule that is used to separate content in an HTML page.
li The li tag defines a list item.
Main The content inside the main element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.
Navigation bar(nav) The nav tag defines a set of navigation links.
Noscript The noscript tag defines an alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script.
Ordered list(ol) The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
Paragraph The p tag defines a paragraph.
Preformatted Text(pre) Text in a pre element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.
section The section tag defines a section in a document.
Table The table tag defines an HTML table.
tfoot The tfoot tag is used to group footer content in an HTML table.
Unordered List(ul) The ul tag defines an unordered (bulleted) list.
video The video tag is used to embed video content in a document, such as a movie clip or other video streams.
